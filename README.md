# Mobile First Login UI

A responsive login interface built with HTML and CSS, following the Mobile First approach.

## Overview

This project demonstrates how to create a clean and responsive login page, starting from mobile devices and scaling up to larger screens.

## Live Preview

https://saulo-lima.github.io/projeto-login/

## Technologies

* HTML5
* CSS3
* Media Queries

## Features

* Responsive design
* Mobile First workflow
* Clean and minimal interface
* No frameworks (pure HTML and CSS)

## Notes

This is a front-end only project. The login functionality is not implemented (no backend).
